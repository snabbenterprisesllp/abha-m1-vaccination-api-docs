# Screenshots Directory

This directory is intended for UI screenshots that demonstrate the Vaccination Locker system functionality for ABDM M1 Sandbox review.

## Recommended Screenshots

If available, please include screenshots of:

1. **Parent Onboarding**
   - Registration screen
   - OTP verification screen
   - Profile completion screen

2. **Child Profile Management**
   - Add child profile form
   - Child profile detail page
   - Birth details section

3. **Vaccination Timeline**
   - Timeline view showing administered vaccines
   - Upcoming vaccinations
   - Status indicators (Administered, Due, Upcoming)

4. **ABHA Linking Flow**
   - ABHA number entry screen
   - Consent form display
   - Consent acceptance confirmation
   - ABHA linking status

5. **Vaccination Records**
   - Vaccination record creation form
   - Vitals capture section
   - Vaccination detail page

6. **Reminders**
   - Reminder settings
   - Notification preferences
   - Upcoming reminders list

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Resolution**: Minimum 1920x1080 for clarity
- **Privacy**: Mask or blur any real personal information
- **Naming**: Use descriptive names (e.g., `abha-linking-consent-form.png`)
- **Annotations**: Add text annotations if needed to explain features

## Note

If screenshots are not available during M1 review, the API documentation and flow diagrams should be sufficient for ABDM reviewers to understand the system functionality.
